# موقع قمة التميز

هذا المستودع يحتوي على ملفات موقع **قمة التميز** المبني باستخدام **Jekyll** والمستضاف على **GitHub Pages**.

## المحتوى
- مقالات متخصصة في الإدارة، الجودة، والحوكمة.
- صفحات ثابتة مثل: من نحن، تواصل معنا، الخدمات.

## الفهرسة
تم تفعيل **IndexNow** لإرسال الصفحات الجديدة والمحدّثة تلقائيًا لمحركات البحث.

## النشر
يتم نشر الموقع تلقائيًا عند كل تحديث عبر GitHub Actions.
